\section{Quantitative Results}\label{sec:exp:results}

This section reports primary clustering scores for the proposed model and for each baseline under the data conditions in \cref{sec:exp:datasets}.
Rows index methods (including ablated variants where applicable); columns index emitter-level and mode-level \gls{nmi}, \gls{ari}, and \gls{acc} after optimal alignment of cluster indices to reference labels where those labels exist only for evaluation (\cref{sec:method:metrics}).
Each entry is aggregated over random seeds and window splits as stated in \cref{sec:exp:setup,sec:exp:hparams}; mean $\pm$ one standard deviation is shown when multiple seeds are available.
