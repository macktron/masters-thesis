\section{Pulse Descriptor Words}\label{sec:bg:pdw}

After detection and parameter estimation in the pipeline of \cref{sec:bg:elint_pipeline}, each accepted pulse is summarised by a \emph{\gls{pdw}}: a fixed-layout record of measured pulse parameters that discards intermediate waveforms while retaining the quantities \gls{elint} algorithms consume \parencite{wiley2006elint}.
Indexing pulses in time order by $n \in \{1,2,\ldots\}$, write the $n$th descriptor as a real vector
\begin{equation}\label{eq:bg:pdw_vector}
  \vect{x}_n
  =
  \bigl(
    f_n,\,
    \mathrm{PW}_n,\,
    t_n,\,
    A_n,\,
    \theta_n,\,
    \ldots
  \bigr)^{\top}
  \in \R^{d},
\end{equation}
where $f_n$ is the measured carrier frequency, $\mathrm{PW}_n$ the pulse width, $t_n$ the \gls{toa} of a chosen reference point on the pulse (often the leading edge), $A_n$ a scalar amplitude or energy proxy, and $\theta_n$ the \gls{aoa} when direction finding is available.
The ellipsis collects any additional fields the sensor supplies (sub-band identifiers, polarisation, intra-pulse modulation labels, and so on); the dimension $d$ is therefore platform-specific but fixed for a given export format once the column order is fixed.
\cref{fig:bg:pdw} illustrates the core scalar fields in a minimal five-dimensional template aligned with \cref{eq:bg:pdw_vector}.

When pulses are already assigned to a single coherent pulse train, successive \glspl{toa} define inter-pulse intervals
\begin{equation}\label{eq:bg:pdw_pri}
  \tau_n = t_n - t_{n-1}, \qquad n \ge 2,
\end{equation}
from which a nominal \gls{pri} is read directly and a \gls{prf} follows as $\mathrm{PRF}_n = 1/\tau_n$ whenever $\tau_n$ is well defined and the train is approximately periodic.
Staggered or jittered \gls{pri} programmes replace a single $\tau_n$ with structured sequences of intervals; in that setting either $\tau_n$ is stored explicitly in the \gls{pdw} stream or stagger parameters appear as additional columns, but the abstract role of \cref{eq:bg:pdw_vector} is unchanged: each index $n$ labels one pulse vector in $\R^{d}$.

Measured descriptors are noisy projections of latent physical state.
A convenient bookkeeping form is $\vect{x}_n = \vect{g}(\vect{s}_n) + \vect{\epsilon}_n$, where $\vect{s}_n$ collects true frequency, timing, and antenna coupling, $\vect{g}$ is the deterministic part of the sensor forward model, and $\vect{\epsilon}_n$ aggregates thermal noise, quantisation, and parameter-estimation error.
Weak pulses yield large effective noise or missing fields; false detections insert spurious samples with no true $\vect{s}_n$.
\emph{Deinterleaving} (\cref{sec:bg:elint_pipeline}) attempts to partition the detection list into trains so that, within each train, \cref{eq:bg:pdw_pri} is interpretable; residual mixture, missed pulses, and extra pulses then appear as structured outliers in the sequence $(\vect{x}_n)$ rather than as isolated scalar errors.
